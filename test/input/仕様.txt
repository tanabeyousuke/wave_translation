!音楽ファイルの内容はこんな感じにする予定
!一大のシンセにreal型のレジスタがr1からr64まである

setting
	osc
		delta 30
		square 70 !オシレータ自体の音量 最大で100,integer型
	end osc

	filter
		lowpass r1 r2 !フィルターの設定 第一にカットオフ周波数、第二にレゾナンス
		highpass r3 r4 !フィルターは複数書いた場合上から直列で接続される
	end filter

	env
		atk r5	!今回は上からアタック、ディケイ、サスティン、リリース
		dec r6
		sus r7
		rel r8
	end env

	lfo
		sin r1 r9 r10 !第一に出力レジスタ、第二に周波数、第三に振幅
	end lfo

	amp
		amp r5 !最終的にどれぐらいの大きさで音を出すか決めるアンプ
	end amp 

end setting

exec
	r10 1.0 !レジスタへ代入 この場合、r10はlfoのsinの振幅につながっているので、振幅が1.0になる
	... !他のも色々代入しなきゃいけないけど、中略
	
	speed 192 !速度記号 この場合、四分音符が一分間に192回入る速さ
	dis 4 4p !音符 この場合、第四オクターブdisの付点四分音符
	rest 1 !休符 この場合、全休符

	r6 3.1 !レジスタの値はいつでも変えられるが、この例で言うところのr1など、lfoの出力などは代入後にすぐ変えられる可能性がある

	... !音符とかをもっと書いてく
	!このソフトでは、音割れを防ぐために実際の出力の値は ampの値/シンセサイザの数にする予定

end exec

!あくまで予定
